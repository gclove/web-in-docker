Archlinux 个人软件源使用帮助

简介

支持架构: i686 x86_64

收录软件: 点击查看

更新历史: 请查看shmilee.log文件

使用说明

1. 以 x86_64 为例, 编辑 /etc/pacman.conf, 在文件最后添加以下条目(操作前请做好相应备份)。

[shmilee]
Server = http://#MyIP#/repo-shmilee/os/x86_64

2. 添加密钥。

密钥 id = E2054497

密钥指纹 = 4B51 EF5E C19C 52C4 6100 8E3A 0109 BD10 E205 4497 

# pacman-key -r E2054497
# pacman-key -f E2054497 # verify the fingerprint
# pacman-key --lsign-key E2054497
